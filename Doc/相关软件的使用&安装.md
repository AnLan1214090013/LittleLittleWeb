# 服务器&操作系统
## 服务器连接
使用`XShell`进行服务器连接，下载xshell后打开，点击增加新连接  
<img src="https://user-images.githubusercontent.com/65151826/233850327-987f2230-6a8d-4d60-8434-931af84e0dbf.png" width="800px">  
在`Name`随便填入名字，在`Host`处填入服务器IP，然后点击`Connect`
